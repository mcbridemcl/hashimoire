Adding vol.py & vol/volshell command to shells
bash (The default shell)

    Open a terminal or SSH session
    Make sure you are in a bash shell. If not, type bash and hit enter
    Type the following commands and press enter after each line (replace username with your actual username)
    echo 'export PATH=/home/username/.local/bin:$PATH' >> ~/.bashrc
    . ~/.bashrc

fish (My personal favorite shell)

    Open a terminal or SSH session
    Make sure you are in a fish shell. If not, type fish and hit enter
    Type the following commands and press enter after each line (replace username with your actual username)
    mkdir -p ~/.config/fish
    echo 'set -x PATH /home/username/.local/bin $PATH' >> ~/.config/fish/config.fish
    . ~/.config/fish/config.fish

ksh or sh

    Open a terminal or SSH session
    Make sure you are in a ksh or sh shell. If not, type ksh or sh and hit enter

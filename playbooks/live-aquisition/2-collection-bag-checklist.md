# Collection Bag Checklist

**Use case:** field readiness | deployment checklist | pre-response validation  
**Environment:** live response | rapid evidence collection | seizure support  
**Risk:** caution  

> **Caution:** A field collection bag should be packed and validated before deployment. Missing a simple item such as the correct cable, storage media, or documentation supplies can reduce the quality of the collection or force avoidable evidence handling mistakes.

---

## Purpose

This checklist identifies the physical items that should be prepared and maintained in a rapid evidence collection bag.

The goal is to ensure the responder arrives with the equipment needed to document the initial system state, collect volatile and targeted evidence, preserve integrity, package items correctly, and transition the collection to evidence storage or lab follow-on work.

---

## Primary field bag categories

A practical collection bag should cover five core areas:

1. Documentation
2. Tool delivery
3. Storage and acquisition
4. Power and adapters
5. Evidence packaging and custody

---

## 1. Documentation items

These items help preserve the state of the machine and environment before any interaction occurs.

Recommended items:

- Camera or approved phone camera device
- Charging cable or battery pack for camera device
- Notebook or field notes pad
- Pens
- Permanent marker
- Printed field checklist
- Printed chain-of-custody form if required
- Printed quick reference sheet for naming conventions
- Time-synced watch or approved time reference

### Why these matter

These items support:

- photographing the screen before touching the device
- documenting open windows and user state
- recording visible peripherals
- recording timestamps, serial numbers, and device identifiers
- preserving notes that explain what was done and when

---

## 2. Tool delivery media

These are the physical media used to bring approved collection tools into the field.

Recommended items:

- Primary USB tool drive
- Backup USB tool drive
- Write-protected or read-only tool media if used by policy
- Clearly labeled tool media case or sleeve

### Preparation notes

- Tool media should be labeled clearly
- Primary and backup media should be tested regularly
- The responder should know what tool categories exist on the drive
- Avoid mixing unrelated files onto tool delivery media

---

## 3. Storage and acquisition media

These are the devices used to save collected artifacts.

Recommended items:

- Primary external SSD for target acquisition
- Backup external SSD
- Protective case for each drive
- Short, known-good cables for each storage device
- Labels for evidence media
- Small storage sleeves for flash drives or removable media

### Preparation notes

- Confirm the media has sufficient free space
- Confirm the media supports large files if memory capture is in scope
- Confirm the cable needed for the device is packed with it
- Keep storage devices labeled by purpose
- Avoid carrying unlabeled drives with unclear prior use

### Typical uses

These drives may hold:

- memory images
- exported logs
- targeted user artifacts
- browser artifacts
- screenshots or photographs
- hash outputs
- notes and case documentation

---

## 4. Power and adapter items

These are the boring heroes of the bag.

Recommended items:

- USB-A to USB-C adapters
- USB-C to USB-A adapters
- USB-C hub
- Spare USB cables
- External drive cables
- Ethernet adapter if needed
- Compact power strip or extension cord if appropriate
- Portable battery pack
- Device chargers for collection hardware when needed
- Known-good mouse if useful and permitted

### Why these matter

Field collections often involve:

- low-battery laptops
- devices with too few ports
- modern devices with only USB-C
- legacy devices that still require older connectors
- storage media that shipped with the one cable now living in a drawer somewhere

---

## 5. Evidence packaging and custody items

These items help support orderly packaging and transfer after collection.

Recommended items:

- Evidence bags or approved packaging
- Tamper-evident seals if required
- Evidence tags
- Asset labels
- Custody forms
- Small envelopes or sleeves for loose storage media
- Protective pouches for collected removable media

### Why these matter

Collection does not end when the file copy finishes. The evidence still has to survive handoff without becoming an unlabeled mystery object.

---

## 6. Optional but useful field items

Depending on policy and mission, additional items may include:

- Disposable gloves if appropriate
- Small flashlight
- Serial number scanner or barcode scanner if used
- Portable clipboard
- Basic cable organizer
- Anti-static bags if handling loose drives or components
- Spare labels
- Printed contact list for escalation or approvals

---

## Bag organization suggestion

A collection bag is easier to use when organized by function rather than by chaos.

Suggested layout:

### Documentation pouch
- camera
- notebook
- pens
- marker
- printed forms

### Tool media pouch
- primary tool USB
- backup tool USB

### Storage pouch
- primary acquisition drive
- backup acquisition drive
- drive cables

### Adapter and power pouch
- hub
- adapters
- spare cables
- battery pack
- power accessories

### Evidence packaging pouch
- bags
- labels
- seals
- custody forms

---

## Pre-deployment validation checklist

Before heading out, verify:

- camera or photo device works
- note-taking supplies are present
- primary USB tool drive is packed
- backup USB tool drive is packed
- primary acquisition drive is packed
- backup acquisition drive is packed
- correct cables are packed
- adapters and hub are packed
- evidence labels and forms are packed
- bag contents match the checklist

---

## Field deployment sequence

At a minimum, the physical bag should support this sequence:

1. Arrive with all necessary supplies
2. Document the device state before touching it
3. Connect approved tool media
4. Connect approved acquisition storage
5. Perform collection
6. Label outputs and notes
7. Package evidence or collected media
8. Transfer to storage or lab with documentation

---

## Notes for field use

A good collection bag should reduce improvisation. The responder should not have to wonder where the camera is, whether there is enough storage available, or whether the only USB-C adapter is still sitting on a desk back at the office.

A boring, complete, well-maintained bag beats a clever but incomplete one every time.
